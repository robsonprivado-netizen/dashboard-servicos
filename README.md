# Dashboard Comercial Serviços 2026

## Como rodar

### 1. Abra o Terminal no Mac
`Command (⌘) + Espaço` → digite `Terminal` → Enter

### 2. Entre na pasta do projeto
```
cd ~/Downloads/dashboard-servicos
```

### 3. Instale as dependências (só na primeira vez)
```
npm install
```

### 4. Rode o dashboard
```
npm run dev
```

O navegador vai abrir automaticamente em: **http://localhost:5173**

---

## Como usar a Análise de IA

1. Role até o final do dashboard
2. Clique em **"◈ Gerar Análise Executiva"**
3. Cole sua API Key da Anthropic quando solicitado
   - Obtenha em: https://console.anthropic.com
4. Clique em **Confirmar** e aguarde a análise

---

## Para parar o servidor
No Terminal, pressione `Ctrl + C`
